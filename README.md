# Hide-YouTube-Shorts
Hide YouTube Shorts on the web using uBlock Origin.

---

## 🇬🇧 English

### 🛠 Introduction
This project was developed with the support of [Claude AI](https://claude.ai/).
The goal is to **hide or remove all Shorts content on YouTube** effectively — on both desktop and mobile web.

### 📌 Installation Guide

#### 1️⃣ Install uBlock Origin
First, install the **uBlock Origin** browser extension from the official store:

- **Chrome Web Store**: [Install here](https://chromewebstore.google.com/detail/ublock-origin/cjpalhdlnbpafiamejdnhcphjbkeiagm)
  > ⚠️ Chrome has disabled Manifest V2 extensions for most users. If uBlock Origin doesn't work after installing, follow the workaround guide here: [chrome-enable-mv2](https://github.com/onlytrisdev/chrome-enable-mv2/tree/main)
- **Firefox Add-ons**: [Install here](https://addons.mozilla.org/en-US/firefox/addon/ublock-origin/)
- **Microsoft Edge Add-ons**: [Install here](https://microsoftedge.microsoft.com/addons/detail/ublock-origin/odfafepnkmbhccpbejgmiehpchacaeak)
- **Other browsers**: Search "uBlock Origin" in your browser's extension store.

#### 2️⃣ Add the YouTube Shorts Block Filter
Once installed, follow these steps:

1. **Copy the filter text** from [copy-me.txt](https://github.com/CheemsGalaxy/Hide-YouTube-Shorts/blob/main/copy-me.txt).
2. Open the **uBlock Origin dashboard** — click the uBlock Origin icon in your browser toolbar, then click the ⚙️ settings icon.
3. Go to the **"My filters"** tab.
4. **Delete** any existing YouTube Shorts filters, if you have any.
5. **Paste** the filter provided by this project.
6. Click **"Apply changes"** to save.

That's it — Shorts should now be hidden across YouTube.

## 🇻🇳 Tiếng Việt

### 🛠 Giới thiệu
Dự án này được phát triển với sự hỗ trợ của [Claude AI](https://claude.ai/).
Mục tiêu là **ẩn/loại bỏ hoàn toàn nội dung Shorts trên YouTube** một cách hiệu quả — trên cả giao diện desktop lẫn mobile web.

### 📌 Hướng dẫn cài đặt

#### 1️⃣ Cài đặt uBlock Origin
Trước tiên, cài extension **uBlock Origin** từ cửa hàng chính thức:

- **Chrome Web Store**: [Cài tại đây](https://chromewebstore.google.com/detail/ublock-origin/cjpalhdlnbpafiamejdnhcphjbkeiagm)
  > ⚠️ Chrome đã vô hiệu hóa extension Manifest V2 với hầu hết người dùng. Nếu sau khi cài mà uBlock Origin không hoạt động, làm theo hướng dẫn khắc phục tại đây: [chrome-enable-mv2](https://github.com/onlytrisdev/chrome-enable-mv2/tree/main)
- **Firefox Add-ons**: [Cài tại đây](https://addons.mozilla.org/en-US/firefox/addon/ublock-origin/)
- **Microsoft Edge Add-ons**: [Cài tại đây](https://microsoftedge.microsoft.com/addons/detail/ublock-origin/odfafepnkmbhccpbejgmiehpchacaeak)
- **Trình duyệt khác**: Tìm "uBlock Origin" trong kho extension của trình duyệt bạn đang dùng.

#### 2️⃣ Thêm filter chặn YouTube Shorts
Sau khi cài xong, làm theo các bước sau:

1. **Copy đoạn filter** từ file [copy-me.txt](https://github.com/CheemsGalaxy/Hide-YouTube-Shorts/blob/main/copy-me.txt).
2. Mở **uBlock Origin dashboard** — bấm vào icon uBlock Origin trên thanh công cụ trình duyệt, rồi bấm biểu tượng ⚙️ (settings).
3. Vào tab **"My filters"**.
4. **Xóa** các filter chặn YouTube Shorts cũ nếu bạn đã có sẵn.
5. **Dán** filter của dự án này vào.
6. Bấm **"Apply changes"** để lưu.

Xong — Shorts sẽ được ẩn trên toàn bộ YouTube.
