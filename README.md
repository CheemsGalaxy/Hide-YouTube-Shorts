# Hide-YouTube-Shorts
Hide YouTube Shorts on the web using uBlock Origin.

---

## 🛠 Introduction
This project was developed with the support of [Claude AI](https://claude.ai/).
The goal is to **hide or remove all Shorts content on YouTube** effectively — on both desktop and mobile web.

## 📌 Installation Guide

### 1️⃣ Install uBlock Origin
First, install the **uBlock Origin** browser extension from the official store:

- **Chrome Web Store**: [Install here](https://chromewebstore.google.com/detail/ublock-origin/cjpalhdlnbpafiamejdnhcphjbkeiagm)
  > ⚠️ Chrome has disabled Manifest V2 extensions for most users. If uBlock Origin doesn't work after installing, follow the workaround guide here: [chrome-enable-mv2](https://github.com/onlytrisdev/chrome-enable-mv2/tree/main)
- **Firefox Add-ons**: [Install here](https://addons.mozilla.org/en-US/firefox/addon/ublock-origin/)
- **Microsoft Edge Add-ons**: [Install here](https://microsoftedge.microsoft.com/addons/detail/ublock-origin/odfafepnkmbhccpbejgmiehpchacaeak)
- **Other browsers**: Search "uBlock Origin" in your browser's extension store.

### 2️⃣ Add the YouTube Shorts Block Filter
Once installed, follow these steps:

1. **Copy the filter text** from [copy-me.txt](https://github.com/CheemsGalaxy/Hide-YouTube-Shorts/blob/main/copy-me.txt).
2. Open the **uBlock Origin dashboard** — click the uBlock Origin icon in your browser toolbar, then click the ⚙️ settings icon.
3. Go to the **"My filters"** tab.
4. **Delete** any existing YouTube Shorts filters, if you have any.
5. **Paste** the filter provided by this project.
6. Click **"Apply changes"** to save.

That's it — Shorts should now be hidden across YouTube.
